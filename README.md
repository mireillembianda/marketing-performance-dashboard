# Dashboard de Performance Marketing Multicanal

## Contexte
Analyse de la performance de campagnes marketing sur plusieurs canaux (Google Ads, Meta, TikTok).

## Objectifs
- Mesurer la rentabilité marketing
- Optimiser l’allocation budgétaire
- Comparer les canaux

## Outils
- Power BI
- DAX
- Power Query

## Analyses
- Modélisation en étoile
- Calcul de KPI : ROI, ROAS, CAC
- Analyse temporelle et géographique

## Résultats
- Identification des canaux les plus rentables
- Optimisation de la répartition budgétaire
- Mise en évidence des campagnes performantes
